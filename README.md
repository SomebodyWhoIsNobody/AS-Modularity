# AS-Modularity
AS-Modularity | Apple Shortcuts - Modularity | An AI agent framework for getting stuff done.

## A Quick Note
Before we get started, I should mention that this is not meant to be a replacement for [Hermes](https://hermes-agent.nousresearch.com/) or [OpenClaw.](https://openclaw.ai/)
It is merely a proof of concept and should not be treated like a professional tool. Security is limited and should not be trusted with important information.
